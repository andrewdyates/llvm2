## 5. Issues Found and Fixed

### Fixed: Non-exhaustive match in call_lowering_proofs.rs (P2)

**File:** `crates/llvm2-verify/src/call_lowering_proofs.rs` lines 1159, 1173
**Problem:** Two test helper functions (`assert_valid`, `assert_invalid`) did not handle the `VerificationResult::Unknown` variant, causing compilation failure when building tests.
**Root cause:** The `Unknown { reason: String }` variant was added to `VerificationResult` (verify.rs:165) but the test helpers in `call_lowering_proofs.rs` were not updated.
**Fix:** Added explicit `Unknown` match arms that panic with descriptive messages.
**Impact:** Blocked `cargo test -p llvm2-verify -- full_proof_suite` and any test build that included this module's tests.
