# Support

## Getting Help

For questions, bug reports, and feature requests, please use **GitHub Issues**:

1. **Bug reports**: Open an issue with the `bug` label describing the issue, steps to reproduce, and expected behavior.
2. **Feature requests**: Open an issue with the `feature` or `enhancement` label describing the desired functionality.
3. **Questions**: Open an issue with the `question` label.

## Security Issues

For security vulnerabilities, **do not** open a public issue. Instead, see [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Communication Channels

- **Primary**: GitHub Issues on this repository
- **Organization updates**: [DashNews](https://github.com/dropbox-ai-prototypes/dashnews/discussions)

## Issue Response

Issues are triaged and prioritized by the AI team. Response times vary based on priority:

| Priority | Typical Response |
|----------|------------------|
| P0 | Immediate |
| P1 | Same day |
| P2 | Within 48h |
| P3 | As capacity allows |

See repository labels for current priority assignments.
